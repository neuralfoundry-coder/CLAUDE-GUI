## 참고

- 본 SRS는 IEEE 830 표준을 기반으로 하되, 실무 활용에 맞게 간소화한 구조를 따른다.
- 기능 요구사항은 `FR-{카테고리}{번호}` 형식, 비기능 요구사항은 `NFR-{카테고리}{번호}` 형식으로 번호를 부여한다.
- 유스케이스는 `UC-{번호}` 형식으로 식별한다.
